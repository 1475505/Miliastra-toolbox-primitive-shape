轮廓描边工具

> 本项目代码完全由 AI 生成。

## 原始需求

输入一张图片和可用的基础图元（可缩放）列表，需要得到根据图元组合拼接来拟合图片的方案。只描边主体的轮廓，不填充，不能过度超越主体图片（到轮廓外）。类似于用"串珠"的效果来拟合图形。

比如输入一颗爱心的曲线，如何通过圆形（可伸缩成椭圆）的拼接来拟合这颗爱心的曲线？

图元：
- 圆形（金币）：可缩放成椭圆
- 正方形：可缩放成矩形、可旋转（可以考虑通过 shapely 的拆分成折线等功能来处理？）
- 自定义矩形
- 三角形

默认这里最简单版本只考虑圆形和矩形，不需要考虑颜色。说明：
1. 可以用图元的不同变换效果，不一定是同样大小的。比如用椭圆的弧线去贴合轮廓。不要有空档。向图片内侧填充，不能超出轮廓。
2. 尽量让椭圆的边弧线去贴合轮廓，而不是对轮廓取一些点变成椭圆。
3. 输出内容：A. 仅图元拼接的效果图 B. 图元拼接效果图和原图叠加对比的图 C. 所有图元的相对坐标，缩放比例，旋转角度列表。 D. 图片中心坐标位置。
4. 图元尽可能地大。比如，不是说对轮廓的每一个点都替换成椭圆，而是用尽量大又贴合轮廓的椭圆去拟合，但保持尽量不超出轮廓。

## 技术方案

最终技术方案请参考[tech.md](tech.md)

提供两类方向：

### 方案1：SVG曲线拟合
1. 使用[VTracer](https://github.com/visioncortex/vtracer)将输入的图片转化成 svg 格式。svg 格式会携带贝塞尔曲线、图元等信息。
2. 根据贝塞尔曲线路径上的点和特性，通过调整图元的缩放，让图元的边（如椭圆弧线）和轮廓重合，并且整个图元不会超出轮廓圈定的范围（主体填充）。
3. 给出图元组合拼接相关的坐标、缩放（可以变小）、旋转（欧拉角）等等。

### 方案2：OpenCV解析点拟合
1. 使用 OpenCV 等库将输入的图片轮廓点集。
2. 类似地。输入图元，将点集的轮廓进行拟合，给出图元组合拼接相关的坐标、缩放（可以变小）、旋转（欧拉角）等等。
